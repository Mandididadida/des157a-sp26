Inheritance in CSS

Inheritance in CSS means that some styles applied to a parent element are automatically passed down to the child elements inside it. This is common with text styles, which helps keep pages consistent. For example, color, font-family, and line-height are often inherited. Properties such as margin, padding, border, width, and background-color are usually not inherited because each element often needs its own box styling.

Specificity in CSS

Specificity is the way CSS decides which rule should be used when more than one rule matches the same element. It is important because it controls which style wins. In general, ID selectors are stronger than class selectors, and class selectors are stronger than element selectors. A compound selector combines selectors without commas, such as nav ul, while multiple selectors use commas, such as h1, h2, p, to apply the same rule to separate elements.

ID and Class Attributes

ID and class attributes are both used to identify elements in HTML so they can be styled in CSS. An ID is meant for one unique element on a page, while a class can be used on many elements. In CSS, an ID is selected with a hash symbol like #header, and a class is selected with a period like .box. Classes are useful when the same style needs to be repeated in several places.
